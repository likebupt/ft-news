### Microsoft Research
- **SkillOpt: Agent skills as trainable parameters** — Microsoft frames agent “skills” (instruction/tool-use behaviors) as trainable parameters that can be optimized from feedback without updating the base model weights. For finetuning, this is a notable post-training pattern: improve agent behavior with a lightweight, safer alternative to full model finetunes. [Read more](https://www.microsoft.com/en-us/research/blog/skillopt-agent-skills-as-trainable-parameters/)
- **Memora: A Harmonic Memory Representation Balancing Abstraction and Specificity** — Memora introduces a scalable agent memory system that separates what is stored from how it is retrieved, aiming to retain both high-level abstractions and specific details over long tasks. This matters for post-training because memory design can materially improve long-horizon agent performance, reducing pressure on context-window scaling or repeated retrieval. [Read more](https://www.microsoft.com/en-us/research/blog/memora-a-harmonic-memory-representation-balancing-abstraction-and-specificity/)

### Apple Machine Learning
- **Amortizing Maximum Inner Product Search with Learned Support Functions** — Apple proposes a learned, regression-based approach to maximum inner product search (MIPS), a core primitive for nearest-neighbor retrieval. It is not a finetuning method directly, but it could make retrieval-heavy post-training workflows cheaper and faster, including data selection, hard-negative mining, and RAG-assisted finetunes. [Read more](https://machinelearning.apple.com/research/amortizing-inner-product-search)

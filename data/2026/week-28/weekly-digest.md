## Microsoft Research
- **Aurora 1.5** extends Microsoft’s open weather foundation model with 22 more variables, hourly temporal resolution, and probabilistic ensemble forecasting. For finetuning, it’s a useful case study in domain-specific post-training that expands label/output space and adds uncertainty-aware behavior for real-world scientific deployment. [Read more](https://www.microsoft.com/en-us/research/blog/aurora-1-5-extending-open-foundation-models-for-weather-and-earth-system-applications/)

## Amazon Science
- **Turnstile** is a new Rust proxy that captures token IDs and other generation metadata during agentic interactions, avoiding the information loss of plain-text transcripts. This matters for RL and post-training because token-level traces improve credit assignment, reward computation, and reproducibility for agent finetuning pipelines. [Read more](https://www.amazon.science/blog/capturing-token-ids-during-agentic-interactions-for-better-reinforcement-learning)

## Apple Machine Learning
- **Weblica** introduces scalable, reproducible training environments for visual web agents, addressing the instability of the live web and the limits of static offline datasets. This matters for finetuning because it can support larger-scale supervised finetuning and online RL for browser agents in controlled environments. [Read more](https://machinelearning.apple.com/research/weblica-visual-web-agents)
- **Unmasking On-Policy Distillation** studies when on-policy distillation helps reasoning models, when it hurts, and how teacher choice and decoding strategy affect outcomes. It is directly relevant to post-training because it offers practical guidance on using dense per-token supervision without degrading reasoning quality. [Read more](https://machinelearning.apple.com/research/unmasking-on-policy-distillation)
- **DynaMiCS** proposes dynamic data mixtures for multi-domain finetuning so models can improve on target domains while preserving performance on constrained domains like safety, instruction following, and general knowledge.
